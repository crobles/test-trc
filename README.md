## Name Project

node tree

## El BACKEND

# archivo .env para el backend

se debe crear el archivo .env para el backend en nestjs con los siguientes parametros:

APP_PORT=3100
APP_ID=node-tree-backend
NODE_ENV=development

# instalación

npm i

# Ejecución

npm run start:dev2

# -------------------------------------------------------

## EL FRONTEDN

# instalación

npm i

# Ejecución

npm run dev

# Para mirar el front

localhost:3000
